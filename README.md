Mini Projects

A collection of small JavaScript projects that I built
while learning and practicing frontend development.

---

Projects

 1. Color Changer
A simple project that changes the background color randomly
every time you click a button. Built to practice DOM manipulation
and event handling in JavaScript.

 2. BMI Generator
A tool that calculates your Body Mass Index based on your height
and weight. It also tells you whether you are underweight, normal,
overweight or obese.

 3. KB Key Detector
A small project that detects which keyboard key you are pressing
and displays the key name and key code on the screen.

 4. Digital Clock
A live digital clock that shows the current time and updates
every second. Built using the JavaScript Date object.

 5. Guess The Number
A fun little game where the computer picks a random number and
you have to guess it. It gives you hints whether your guess
is too high or too low.
